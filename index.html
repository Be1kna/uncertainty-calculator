  • Addition/Subtraction: Add absolute uncertainties directly. Use decimal-place precision when rounding.
    • 
                        
  • Multiplication/Division: Add relative uncertainties, then convert to absolute. Use significant-figure rounding.
    • 
                        
  • Mixed operations: If expression contains both mult/div and add/sub, multiplication/division rules take precedence for rounding (use significant figures).
    •
